About Traffic Fines Data

Data Sources

This project visualises data related to traffic offences collected from Australian government sources. The dataset includes:

  • Speeding fines reported by state transport and road safety departments.
  • Mobile phone use while driving violations obtained from police enforcement records.
  • State-level demographic context integrated from Australian Bureau of Statistics (ABS) for age group comparisons.

Data Collection Methods

Traffic offence data is collected through both manual and automated enforcement mechanisms. These include:

  • Police-issued fines: Officers manually record offences such as mobile phone use, speeding, and dangerous behaviour.
  • Fixed and mobile detection cameras: Devices placed on roads to capture speeding and phone violations automatically.
  • Red light and mobile phone detection systems: Used increasingly in urban areas to catch distracted driving.

The dataset includes information on which detection method was used for each offence.

Data Coverage and Updates

The dataset spans from 2008 to 2023 and is updated annually. Each record includes:

  • Year of offence
  • State or territory where the offence occurred
  • Detection method (Police or Camera)
  • Fine amount and count
  • Offender’s age group

This allows for trend analysis over time, comparison between jurisdictions, and insights into enforcement strategies.

How the Dashboard Uses the Data

The dashboard provides two main views — one for Speed Fines and one for Mobile Phone Use. Users can explore the data interactively using:

  • Year slider to view fines in a specific year.
  • Checkbox filters to include or exclude detection methods.
  • Age group buttons to compare behaviour across demographic segments.

Visualisations include pie charts, bar charts, line charts, choropleth maps, and detailed summary tables. These update in real time based on the selected filters.
